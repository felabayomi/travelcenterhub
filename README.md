# Travel Center Hub

A simple travel website with the following main pages:
- Home
- Flights
- Hotels
- Vacation Packages
- Cruises
- Book Travel
- Contact

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Start the development server:
   ```
npm run dev
   ```
3. Open your browser at [http://localhost:5173](http://localhost:5173)

## Project Structure
- `src/` - React source code
- `public/` - Static HTML
